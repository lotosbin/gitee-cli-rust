# gitee-cli-rust

## Install

```shell
cargo install gitee
```

## Usage

```shell
gitee -h
```

```
Usage: gitee <COMMAND>

Commands:
  home  open home page in browser
  help  Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version

```
